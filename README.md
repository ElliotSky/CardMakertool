## 桌游卡片制作器（CardMakertool）

### 项目简介

这是一个基于浏览器的**桌游角色卡片制作工具**，使用原生 HTML / CSS / JavaScript 实现，无需后端和打包构建。  
打开网页即可可视化编辑角色信息与技能，并导出为图片或 JSON 数据，适合桌游原型设计、数值平衡讨论和教学演示等场景。

### 主要功能

- **角色基础信息**
  - 角色名称、卡牌编号、卡牌系列等字段实时编辑
  - 支持上传角色配图，配图区高度可调
- **卡牌外观与配色**
  - 多套内置配色模板（如深曜黑、墨绿黑等）
  - 支持本地保存自定义模板、重命名、右键用当前配色覆盖模板
  - 主色 / 辅助色 / 点缀色一键应用到整张卡片
- **文字样式控制**
  - 角色名称、技能标题、技能名称、技能描述、消耗标签、卡牌编号、卡牌系列等分别配置：
    - 字号、字体、是否加粗
    - 缩放比例（用于绕过浏览器最小字体限制）
- **卡牌布局与尺寸**
  - 自定义卡牌宽高与配图框高度
  - 预设比例：标准卡牌 63:88、2:3、3:4、1:1、4:3 等
  - 内容间距可调
- **技能编辑系统**
  - 动态添加 / 删除技能
  - 支持技能名称、描述、消耗标签等字段
  - 多个技能区域自动排版显示在预览卡片上
- **预览与导出**
  - 正面 / 背面切换预览（Tab 按钮）
  - 预览缩放调节（50%–150%）
  - 基于 `html2canvas` 一键导出卡片截图（PNG）
- **数据导入导出**
  - 导出当前卡片配置为 JSON 文件
  - 从 JSON 导入，快速还原或切换角色配置

### 目录结构（本工具相关）

- **`cardmaker.html`**：主页面，包含编辑面板与卡片预览区
- **`javascript.js`**：业务逻辑脚本（配色模板、技能编辑、预览更新、导入导出、截图等）
- **`style.css`**：整体样式与卡片视觉风格
- （可选）**`../国家博弈data/角色卡/`**：存放导出的 JSON 与图片资源的示例目录

### 使用方式

- **环境要求**
  - 现代浏览器：推荐最新版 Chrome / Edge
  - 无需 Node / Rust / 构建工具，直接本地打开即可

- **本地打开**
  - 方式一：在文件管理器中双击 `cardmaker.html`，用浏览器打开
  - 方式二：将整个 `CardMakertool` 目录放到任意静态服务器（如 nginx、VSCode Live Server）下访问

- **基本操作流程**
  1. 打开 `cardmaker.html`
  2. 在左侧填写角色名称、编号、系列，上传配图
  3. 选择或调整配色模板，微调文字样式和卡牌尺寸
  4. 添加技能条目，编辑技能名称、描述和消耗
  5. 在右侧预览区切换正面 / 背面、调整预览缩放
  6. 根据需要：
     - 点击「导出 JSON」保存配置，方便下次导入或批量管理
     - 点击「导出图片」生成卡牌截图，用于打印或文档展示

### 与仓库其它目录的关系
